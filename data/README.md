# Data Directory

Place your documents here for ingestion into the multi-hop research agent.

## Supported Formats
- PDF files
- CSV files  
- Markdown files
- Text files

## Example Documents
Add documents like:
- `company_x_earnings.pdf`
- `supply_chain_analysis.md`
- `nasdaq_prices.csv`

Run `python ingest.py` after adding documents to index them.
